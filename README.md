# AS OF 1.2 THIS MOD NO LONGER REQUIRES [CAVERNS AND CHASMS](https://modrinth.com/mod/caverns-and-chasms) AND BY PROXY [BLUEPRINT](https://modrinth.com/mod/blueprint) TO FUNCTION!
This is a standalone mod that includes the crumbling effect and functionality from my other mod [Valcon](https://modrinth.com/mod/valcon).

When a configurable list of mobs or projectiles hurts another entity or player, they will inflict the crumbling effect which reduces their armor points by a configurable percentage, for a configurable amount of time, and will optionally display *custom* particles and or play a sound when the effect is over. The following config options and their default values can be found in: `config/crumbling_kunai-common.toml`

Crumbling inflictors: `caverns_and_chasms:kunai`

Crumbling armor penalty: `50%` 

Crumbling length: `10 Seconds`

Crumbling particles: `True`

Crumbling icon: `True`

Crumbling sound effect: `True`
