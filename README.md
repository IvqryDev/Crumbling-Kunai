This is a standalone mod that includes the crumbling effect from my other core mod Valcon. Throwing a kunai at a mob will reduce their armor points by a set percentage, for a set amount of time, and then play a sound effect when that effect has worn off.

The following details and their default values can be configured in config/crumbling_kunai.toml
```Crumbling armor penalty: 50%```
```Crumbling length: 8 Seconds```
```Crumbling particles: False```
```Crumbling sound effect: True```
